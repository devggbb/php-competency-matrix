# Architecture & System Design

Охватывает архитектурное мышление: проектирование сервисов, контрактов и API, выбор архитектурных стилей, интеграции,
шины событий, надёжность, отказоустойчивость, масштабирование, trade-offs, документирование архитектурных решений.
Фокус: умение проектировать системы, которые выдерживают рост нагрузки, изменчивость требований и длительный жизненный
цикл.

## Подкомпетенции

- Основы архитектурного мышления
- Архитектурные паттерны (слои, модули, монолит, микросервисы, CQRS, Event-Driven)
- Проектирование API и контрактов
- Паттерны интеграции и взаимодействия сервисов
- Хранение данных и архитектура данных (storage patterns)
- Надёжность, отказоустойчивость и масштабирование
- Производительность и высоконагруженные системы
- Архитектурные компромиссы и принятие решений
- Документирование архитектуры
