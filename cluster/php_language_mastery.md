# PHP Language Mastery

Покрывает глубину владения самим языком PHP: синтаксис, особенности поведения, типизация, SPL, стандартная библиотека,
OOP-механики, работа с ошибками, стабильность публичного API, безопасность языка, внутренности PHP и его рантайм.
Фокус: способность писать надёжный, чистый, идиоматичный код и эффективно использовать все возможности PHP.
