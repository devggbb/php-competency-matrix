# Framework & Ecosystem

Отражает умение работать с современным PHP-стеком: Symfony/Laravel, Composer, PSR-ы, DI-контейнеры, middleware, модули,
пакеты, SDK, шаблоны архитектуры на уровне фреймворка, интеграции, очередь задач, кэш, тестирование.
Фокус: практическая разработка production-сервисов на выбранном фреймворке и уверенное владение экосистемой библиотек.

## Подкомпетенции

- Знание основ фреймворков (Laravel / Symfony / Yii / др.)
- Понимание жизненного цицила приложения и основных абстракций
- Работа с экосистемными инструментами (Composer, DI-контейнеры и пр.)
- Работа с модулями, пакетами и SDK
- Расширение и кастомизация фреймворков
- Оптимизация работы фреймворков
- Создание собственных библиотек и модулей
