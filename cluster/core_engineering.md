# Core Engineering

Отражает фундаментальные инженерные знания: алгоритмы, структуры данных, сетевое взаимодействие, основы вычислительной
модели, работу памяти и CPU, принципы производительности, конкурентность.  
Фокус: прочная CS-база, умение понимать, как программы работают «под капотом».

## Подкомпетенции

- Алгоритмы
- Структуры данных
- HTTP, TCP/IP
- Concurrency basics
- Memory & CPU понимание работы программ
