# Asynchronous Systems

Этот кластер отражает способность инженера работать с асинхронностью, очередями, событиями, воркерами, потоками и
распределёнными асинхронными системами.

## Подкомпетенции

- Основы асинхронности и конкурентности
- Event loop, async/await, неблокирующий I/O
- Очереди и брокеры сообщений (Kafka, RabbitMQ, NATS, SQS)
- Проектирование асинхронных процессов и воркеров
- Идемпотентность, ретраи, дедубликация событий
- Паттерны распределённых асинхронных систем
- Надёжность, масштабирование и мониторинг асинхронных систем
